# 常用工具集合

> 本工具支持常见字符串、数组操作以及相关校验。

## 字符串工具

类方法`StringUtils.php`

字符串操作提供以及方法：

1. 随机字符串方法：
2. 驼峰命名转下划线命名；
3. 字符串脱敏操作；

## 数组工具

类方法`ArrayUtils.php`；

提供如下方法：

1. 递归获取数组值，支持多维数组
2. 数组合并操作，支持多维数组
3. 数组转换为实例对象，根据反射创建对象

## 公共函数

`common_functions.php`根据工具写的一些快捷方法，便于使用。

1. 手机号脱敏；
2. 用户名称脱敏；

## 验证函数

`validator_function.php`主要包含一些常见数据数据验证方法

1. 手机号验证方法；
2. 身份证号码有效性校验；
